Your 7-Day Learning Journey

From Git basics to advanced collaboration workflows.

Day 1

Git Fundamentals

Understand version control, install Git, configure settings, and create your first repository.

  • What is version control?
  • Installing and configuring Git
  • Initializing a repository (git init)
  • Understanding the Git workflow
Duration: 45 minutes | 3 exercises
Day 2

Basic Git Commands

Learn essential commands: add, commit, push, pull. Understand the staging area and commit best practices.

  • Staging changes (git add)
  • Committing changes (git commit)
  • Viewing history (git log)
  • Working with remote repositories (git push, git pull)
Duration: 40 minutes | 5 exercises
Day 3

GitHub Interface

Navigate the GitHub website, create repositories, manage issues, and use project boards for task tracking.

  • Creating repositories on GitHub
  • Writing effective READMEs
  • Managing issues and labels
  • Using GitHub Projects boards
Duration: 50 minutes | 4 exercises
Day 4

Branching & Merging

Understand the power of branching. Create, switch, merge branches, and learn to resolve merge conflicts.

  • Creating and switching branches (git branch, git checkout)
  • Merging branches (git merge)
  • Understanding merge strategies
  • Resolving merge conflicts
Duration: 55 minutes | 6 exercises
Day 5

Collaboration Workflows

Learn professional team collaboration using Pull Requests, code reviews, forking, and contributing to projects.

  • Understanding Pull Requests
  • Conducting and responding to code reviews
  • Forking repositories
  • Contributing to open source
Duration: 60 minutes | 4 exercises
Day 6

Project Management

Explore GitHub Actions for automation, manage releases, and understand team permissions and security best practices.

  • Introduction to GitHub Actions (CI/CD)
  • Creating releases and tags
  • Managing team permissions
  • Repository security settings
Duration: 50 minutes | 5 exercises
Day 7

Portfolio Project & Certification

Apply your skills by setting up a professional portfolio repository, contributing to a project, and completing the certification assessment.

  • Create a professional portfolio on GitHub
  • Contribute to a simulated open-source project
  • Set up a basic CI workflow
  • Final assessment & Certificate
Duration: 90 minutes | Final Project
